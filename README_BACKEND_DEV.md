# Backend Dev (HUBEX)

## Start the backend (PowerShell)

From the repo root:

`
.\scripts\dev-backend.ps1
`

Optional port:

`
.\scripts\dev-backend.ps1 -Port 8001
`

## Start the backend (bash)

`
./scripts/dev-backend.sh
`

Optional port:

`
./scripts/dev-backend.sh 8001
`

## Important

Backend is Python; do not run npm here.
